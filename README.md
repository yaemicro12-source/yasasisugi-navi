# Yasashisug Navi

Yasashisug Navi は、ADHD傾向がある人・方向音痴の人・移動に不安がある人のための、優しすぎるナビのMVPです。

単なる道案内ではなく、「予定に間に合うために、いつ起きて、いつ家を出ればいいか」をやさしく逆算します。責めない、急かしすぎない、でも必要な時間ははっきり伝えることを大切にしています。

## MVPでできること

- 予定時刻から逆算して、起きる時間・家を出る時間を教える
- 歩く速さに合わせて移動時間を計算する
- 迷いやすい人のために予備時間も含めて計算する
- 目的地、予定開始時刻、到着したい余裕時間、準備時間、朝ごはん時間、予備時間を入力できる
- 移動距離と歩行速度から移動時間を計算する

## ターゲットユーザー

- 初めて行く場所で不安になりやすい人
- 方向音痴の人
- ADHD傾向があり、逆算スケジュールが苦手な人
- 予定前に何時に起きればいいか迷いやすい人

## 使い方

1. `index.html` をブラウザで開きます。
2. 目的地、予定開始時刻、何分前に着きたいかを入力します。
3. 移動距離と歩行速度を入力します。
4. 準備時間、朝ごはん時間、迷った時の予備時間を入力します。
5. 起きる時間、家を出る時間、到着したい時間、移動時間の目安が自動で表示されます。

## 計算の考え方

```text
移動時間 = 移動距離 ÷ 歩行速度
到着したい時間 = 予定開始時刻 - 何分前に着きたいか
家を出る時間 = 到着したい時間 - 移動時間 - 予備時間
起きる時間 = 家を出る時間 - 準備時間 - 朝ごはん時間
```

## 今後の予定

- ルートから外れたら優しく通知する
- 迷ったら写真や見えるものから現在地確認を助ける
- GPSや地図サービスと連携する
- 出発前に段階的なリマインダーを出す
- 「そろそろ靴を履く」「鍵を持つ」など、出発前チェックリストを追加する

## 開発メモ

このMVPは HTML/CSS/JavaScript だけで動きます。ビルドやインストールは不要です。
