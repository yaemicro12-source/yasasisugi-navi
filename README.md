# Yasasisugi Navi

Yasasisugi Navi は、ADHD傾向がある人、方向音痴の人、移動に不安がある人のための「やさしすぎるナビ」MVPです。

単なる経路案内ではなく、「予定に間に合うために、いつ起きて、いつ家を出ればいいか」をやさしく逆算します。急かしすぎず、でも必要な時間ははっきり見えることを大切にしています。

## 現在のMVPでできること

- ナビゲーターキャラがいるメイン画面から、必要な操作を選べる
- メイン画面から、ゴール設定ページと予定確認ページへ移動できる
- 目的地と予定開始時刻を入力できる
- 早めに到着したい分数を入れて、到着目標時刻を計算できる
- 移動距離と歩く速さから、移動時間を計算できる
- 準備時間、朝ごはん時間、迷った時の予備時間を含めて、家を出る時間と起きる時間を逆算できる
- 「迷った」ボタンから、今後追加予定の現在地確認機能の案内を表示できる
- 逆算結果をタイムラインで確認できる
- タイムラインの各予定をチェックして、今日の達成度を確認できる
- アラームON/OFFの見た目と状態切り替えを試せる

## メイン画面の構成

- 現在時刻を大きく表示
- 最初の予定の5分前の時刻を小さく表示
- 画面中央に、ナビゲーターキャラの表示エリアを配置
- 現在は丸いプレースホルダーを仮置き
- キャラの上に吹き出しを表示
- 画面下部に大きめのボタンを3つ横並びで配置
  - もくてきち
  - スケジュール
  - 迷った

今後、キャラクター画像が決まったら、プレースホルダー部分を画像に差し替える予定です。白いキャラ、水色のキャラのどちらでも見やすいように、背景は淡い水色から白のグラデーションにしています。

## ページ構成

- `index.html`: 現在時刻、ナビゲーターキャラ、メインボタンを表示するメイン画面
- `goal.html`: 目的地、予定開始時刻、早めに着きたい分数を入力するもくてきちページ
- `schedule.html`: 移動や準備時間を入力し、タイムラインと達成度を確認する予定確認ページ

## 使い方

1. `index.html` をブラウザで開きます。
2. メイン画面で「もくてきち」を押します。
3. 目的地、予定開始時刻、何分前に着きたいかを入力します。
4. 「スケジュール」を押して予定確認ページへ移動します。
5. 必要に応じて、移動距離、歩く速さ、準備時間などを入力します。
6. タイムラインの丸いチェックを押すと完了扱いになり、達成度が更新されます。
7. アラームON/OFFは今は見た目だけですが、クリックで状態を切り替えられます。

## 計算の考え方

```text
移動時間 = 移動距離 ÷ 歩行速度
到着目標時刻 = 予定開始時刻 - 早めに着きたい時間
家を出る時間 = 到着目標時刻 - 移動時間 - 迷った時の予備時間
起きる時間 = 家を出る時間 - 準備時間 - 朝ごはん時間
```
